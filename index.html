Portfolio

Pick n' Chill

With a team, we a movie app that allows users to filter through movies by name, genre or keyword then displays the streaming platforms the movie is available on. Users have the option to add the movie to their watchlist and post on the community board about the movie. This app must be downloaded as it uses a Node.js and Express server. Other technologies used include HTML, CSS, Boostrap, JavaScript, jQuery, MySQL, Sequelize, the TMDB and Utelly APIs, Flickity and Animate On Scroll (AOS)

Hitchhikers Travel App

With a team, we conceived and executed a one-stop shop travel app that solves the problem of having to use multiple sites to plan a trip by integrating data received from multiple server-side API requests. This app also uses HTML, CSS, Bootstrap, JavaScript and jQuery.

Bamazon Store

Bamazon is the low-budget version of Amazon that can be used to pretend to purchase common items by taking in orders from users and depleting stock from the store’s MySQL database inventory. The app must be downloaded as it runs in the terminal using JavaScript, jQuery and the Inquirer JSON package.

GifTastic

In this assignment, GIPHY API was used to make a dynamic web page that populates with gifs of the user’s choice using HTML, CSS, JavaScript, jQuery and AJAX calls.

Halloween Trivia Game

The Halloween Trivia Game is a game used to check the users Halloween knowledge (or ability to correctly guess). The user will be given multiple questions to answer and at the end of the game will receive their total score. This game uses HTML, CSS, JavaScript and jQuery.

Psychic Game

The Psychic Game is a guessing game in which the user will have eight chances to guess a randomly generated letter. This app runs in the browser, and features dynamically updated HTML and CSS powered by JavaScript.

Crystal Collector Game

The Crystal-Clicker-Game is a randomized number guessing game. The user will be given a goal number to reach and four crystals with randomized values to use to reach the goal number. If the user surpasses the goal number, they lose. This project was created using HTML, CSS, JavaScript and jQuery.
